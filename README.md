Flight Management System
This is a Flight Management System project coded in C language using the Dev C IDE. The system utilizes file storage for data management.

Features
Sign-up: Allows users to create an account for accessing the system.
Booking Flights: Enables users to book flights to various destinations.
Flight Categories: Provides different categories of flights (e.g., economy, business class).
Calculations: Performs calculations related to flight bookings and billing.
Installation

Clone the repository:
bash

Open the project in Dev C IDE and compile the main source file.

Run the executable:
Execute the compiled binary file to start the Flight Management System.

Usage
Sign-up: Upon starting the system, users can sign up with their credentials.
Booking Flights: Users can search for flights and make bookings for their desired destinations.
Flight Categories: The system provides different flight categories for users to choose from.
Calculations: The system calculates the total booking amount based on selected flights and categories.
